# Coarse Materials（粗料）

本目录存放按项目/公司整理后的粗料。粗料是从原材料（raw）中初步提取的事实草稿，回答三个核心问题：

1. 这个项目是什么？（What）
2. 候选人做了什么？（Contribution）
3. 有什么可量化的结果？（Impact）

## 加工标准

- **核心开发者定位**：所有项目中候选人均为绝对核心开发，表述使用「主导」「独立设计并开发」「核心开发 / 架构设计」「从 0 到 1 搭建」。
- **去除 Why / How / Trade-off**：技术实现细节留给面试。
- **标注来源**：每个事实尽可能标注来自哪个 raw 文件或仓库。
- **标注缺失**：不清楚或待补充的事实明确列出，不虚构。
- **简历权重**：每个粗料文件末尾标注 P0/P1/P2/P3，指导后续简历篇幅分配。
