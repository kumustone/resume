## 项目目标（一句话）
接收图片、PDF、Office 文档，基于视觉模型 + 文本分析识别特殊类别内容，输出敏感/涉密信息疑似度评分。

## 候选人贡献
- 设计并实现文件内容识别与敏感检测服务。
- 支持图片、PDF、Office 等多格式文件解析。
- 基于视觉模型与文本分析，识别化工 CAD 图等特殊类别及敏感/涉密信息。
- 提供 HTTP API 服务化部署，支持同步/异步检测。
