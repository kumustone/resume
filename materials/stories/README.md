# stories

本目录存放可复用的面试故事（Story）。

## 格式

每个故事采用 STAR 结构：

```markdown
## 故事标题

- **Situation（背景）**：当时面临什么问题？
- **Task（任务）**：你的目标是什么？
- **Action（行动）**：你做了什么？
- **Result（结果）**：产生了什么影响？

**对应项目**：
**对应简历 bullet**：
**适用面试问题**：
```

## 原则

- 故事基于 `facts/` 和 `evidence/` 中的真实经历。
- 一个故事可以对应多个简历 bullet 和面试问题。
- 技术细节可以在故事中展开，但简历中只保留一句话。
