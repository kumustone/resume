## 一、年度核心交付

### 1. 新一代 API 数据识别脱敏引擎（r_data_service）

**核心成果**：

- 完成核心安全能力的**代际升级**，显著提升对复杂数据场景（多标签、半结构化/非结构化数据）的覆盖能力
- 核心能力实现**多场景统一复用**：网关 / Web / HttpParse / 离线 SDK / AI 大模型网关，减少重复建设与维护成本
- **性能提升**：核心识别与脱敏能力性能提升 **5x+**，整体服务性能提升 **1x+**

### 2. API 网关主线演进（B8 530/830 版本）

- 完成 B8 530/830 版本，主线 R100、R300 发布
- **账号发现功能重新开发**：覆盖度从 70% 提升到 **90% 以上**
- **新增文件脱敏支持**：DOC / PPT / EXCEL / TXT 文件的脱敏能力
- 重构关键基础组件，提升极端场景下的稳定性与可控性

### 3. AI 应用防护墙

- 完成 **AI 防火墙**开发
- 集成大模型防护、聊天聚合脱敏等能力接入

### 4. 新一代安全网关 Nexis（基于 Pingora）

- 基于 **Pingora**（Cloudflare 开源）开发新一代 Rust 网关 Nexis
- 目标：替换当前 Nginx + 外部服务的模式
- 解决多进程模块缓存占用高的问题
- 在 Http Resp 阶段异步接入三方业务，脱敏水印功能下移

**性能对比**（初步测试）：

| 指标 | 对比 Nginx + 脱敏服务 | 提升 |
|------|----------------------|------|
| QPS | +138% | 2.38x |
| RT | -63% | 降低至 37% |
