## 四、可复用素材

### 技术决策

- **Rust + JNI 架构选择**：在需要与 Java 生态集成的场景下，Rust 通过 JNI 调用 SO 的方式，既保留了 Rust 的性能和安全性，又能复用现有 Java 基础设施。这是一个重要的技术选型决策，可作为面试中"技术选型依据"的素材。

### 项目管理

- **工作量估算教训**：低估兼容旧版本和边界功能的工作量，是常见的项目延期原因。后续改进：更细致的拆解、预留缓冲时间、兼容层单独评估。
