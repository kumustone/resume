# prompts

本目录存放 Claude Code 的系统提示词与任务模板。

## 文件规划

| 文件 | 说明 |
|------|------|
| `system-prompt.md` | 首席职业叙事架构师系统提示 |
| `phase-1-analyze.md` | Phase 1：分析当前简历 IA 问题 |
| `phase-2-branch-design.md` | Phase 2：设计 Resume Branch |
| `phase-3-outline.md` | Phase 3：输出整体目录 |
| `phase-4-summary.md` | Phase 4：修改 Summary |
| `phase-5-core-skills.md` | Phase 5：修改 Core Skills |
| `phase-6-project.md` | Phase 6：逐家公司/项目修改 |
| `review-checklist.md` | Resume Review Checklist 执行模板 |

## 原则

- 所有提示词必须引用 `PRD.md` 中的原则。
- 提示词应明确约束：不改变事实、不夸大、不增加不存在的内容、不确定时先提问。
