# outputs

本目录存放 Career Communication System 的输出产物。

## 文件规划

| 文件 | 说明 |
|------|------|
| `resume-security.yaml` | Resume A — Application Security Engineer 数据源 |
| `resume-security.html` | Resume A — HTML 打印版 |
| `resume-security.pdf` | Resume A — PDF 最终版 |
| `resume-backend.yaml` | Resume B — Senior Backend Engineer 数据源 |
| `resume-backend.html` | Resume B — HTML 打印版 |
| `resume-backend.pdf` | Resume B — PDF 最终版 |
| `linkedin.md` | LinkedIn Profile 内容 |
| `self-intro.md` | 自我介绍（中英文） |
| `interview-stories.md` | 面试故事集 |

## 原则

- `outputs/` 是 **Generated**，可以从 `data/` 和 `materials/` 重新生成。
- 不应手动直接修改 outputs 中的内容；修改应回溯到 Source 或 Knowledge Base。
