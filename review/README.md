# review

本目录存放 Resume Repository 的 Review 记录与决策日志。

## 文件规划

| 文件 | 说明 |
|------|------|
| `decisions.md` | 重要设计决策记录（如项目合并/独立、数据取舍、关键词策略） |
| `changes.md` | 每次修改的变更日志 |
| `questions.md` | 向候选人提问及确认结果 |
| `checklist-records/` | 每次 Review Checklist 的执行记录 |

## 原则

- 所有重大修改都应在 `decisions.md` 或 `changes.md` 中留下记录。
- 记录应包含：决策内容、原因、时间、涉及文件。
- 便于长期维护和回溯。
